CS 184: Computer Graphics and Imaging, Spring 2019

Final Project: Choreographed Particle Simulation

Yusuf Fateen, Brian Levis, Jayanth Sundaresan


Particles

We implement a simple partiicle interaction rule, optimized by dividing the screen into a grid using the minimum interaction radius. At each timestep, particle pairwise forces are calculated and particles are moved.

Rendering

Currently, frames are rendered as fast as particles are updated, which is determined by the input frequency. This pipeline is a bit limiting, and could be optimized.

Audio

A python script converts an audio file into an extremely downsampled (low frequency and bitrate) magnitude signal, and saves the file. Our simulator loads this data file into memory, and uses a shell command to play the corresponding audio. The start time is recorded immediately following this call, to keep the simulation in sync.
